# data-mining-course

Матеріали для курсу "Вступ до інтелектуального аналізу даних" (Introduction to Data Mining).

## Встановлення

1. Встановіть Python (якщо він не встановлений)
2. Встановіть залежності:

```bash
pip install -r requirements.txt
```

## Робота з Jupyter Notebook

Після встановлення залежностей запустіть Jupyter Notebook:

```bash
jupyter notebook
```

Або Jupyter Lab:

```bash
jupyter lab
```

Відкрийте потрібний notebook у середовищі Jupyter.
